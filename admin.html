Qué define una posición

Una posición agrupa landing pública, prompt de análisis de CV, prompt de análisis de entrevista y la definición del test (bloques + preguntas). Cuando creas una nueva posición puedes duplicar los prompts y preguntas de la HoE actual y editarlos. El slug determina la URL pública (/positions/<slug>). Para que una posición sea seleccionable por headhunters tiene que tener Compartir con headhunters activado.
